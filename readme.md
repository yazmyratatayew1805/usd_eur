# Test Project for get currencies

## Project installation
```
git clone https://github.com/yazmyratatayew1805/usd_eur.git
composer install
composer update
```
## Project Database installation
You must create database "db_name" and add to .env
```
php artisan migrate
```
## Project run
```
php artisan serve
```


> NOTE: `Valutas` table.


| Field Name | Type             |
|------------|------------------|
| valuta     | String           |
| code       | String           |
| procent    | Integer          |
